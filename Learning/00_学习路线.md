## 3. 第一阶段：基础入门

### 3.1 HTML 基础（1-2天）

**学习目标：**
- 理解 HTML 是网页的结构
- 掌握常用标签：div, p, span, a, img, ul, li, form, input, button
- 理解语义化标签：header, nav, main, footer, article, section

**重点知识点：**
```html
<!-- 基本结构 -->
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>页面标题</title>
</head>
<body>
  <h1>标题</h1>
  <p>段落</p>
  <a href="https://example.com">链接</a>
  <img src="image.png" alt="图片">
</body>
</html>

<!-- 表单 -->
<form action="/submit" method="POST">
  <input type="text" name="username">
  <input type="password" name="password">
  <button type="submit">提交</button>
</form>
```

### 3.2 CSS 基础（2-3天）

**学习目标：**
- 理解 CSS 是网页的样式
- 掌握选择器：标签选择器、类选择器、ID选择器、组合选择器
- 掌握盒模型：margin, padding, border, content
- 掌握布局：flex 布局、grid 布局
- 理解响应式设计：media query

**重点知识点：**
```css
/* 选择器 */
.tag { color: red; }           /* 类选择器 */
#id { color: blue; }          /* ID选择器 */
div p { color: green; }       /* 后代选择器 */

/* 盒模型 */
.box {
  margin: 10px;
  padding: 20px;
  border: 1px solid #ccc;
}

/* Flex 布局 */
.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 10px;
}

/* 响应式 */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
```

### 3.3 JavaScript 基础（5-7天）

**学习目标：**
- 理解 JavaScript 是网页的行为
- 掌握变量、数据类型、运算符
- 掌握条件语句和循环
- 掌握函数定义和调用
- 理解 DOM 操作
- 理解事件处理

**重点知识点：**
```javascript
// 变量
let name = '张三'
const age = 25

// 数据类型
let str = '字符串'
let num = 123
let bool = true
let arr = [1, 2, 3]
let obj = { name: '张三', age: 25 }

// 条件
if (age >= 18) {
  console.log('成年')
} else {
  console.log('未成年')
}

// 循环
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i])
}

// 函数
function greet(name) {
  return '你好,' + name
}

// DOM 操作
const element = document.getElementById('app')
element.textContent = 'Hello'

// 事件
button.addEventListener('click', function() {
  alert('点击了')
})
```

## 4. 第二阶段：框架学习

### 4.1 Vue 3 核心（7-10天）

**学习目标：**
- 理解 Vue 的响应式原理
- 掌握 Composition API
- 掌握常用指令：v-if, v-for, v-bind, v-on, v-model
- 理解组件化开发思想

**学习文档：**
- 阅读：`01_Vue3基础知识.md`

**核心代码：**
```vue
<script setup>
import { ref, reactive, computed, onMounted } from 'vue'

// 响应式数据
const count = ref(0)
const user = reactive({ name: '张三', age: 25 })

// 计算属性
const doubleCount = computed(() => count.value * 2)

// 方法
function increment() {
  count.value++
}

// 生命周期
onMounted(() => {
  console.log('组件挂载完成')
})
</script>

<template>
  <div>
    <p>{{ count }}</p>
    <p>{{ doubleCount }}</p>
    <p>{{ user.name }}</p>
    <button @click="increment">+1</button>
  </div>
</template>
```

## 8. 常见问题

### Q1：学完基础后，是先学 Vue2 还是直接学 Vue3？

**A：** 直接学 Vue 3。Vue 3 是未来主流，性能更好，语法更现代。

### Q2：需要深入学习 JavaScript 底层原理吗？

**A：** 先学会用，再学原理。可以先跳过原型链、闭包等高级内容，等有需求时再深入。

### Q3：学完这些能找到工作吗？

**A：** 这套路线覆盖了 Vue 全栈开发的基础，可以胜任 Vue 初级/中级开发岗位。但还需要多练项目、深入理解原理。

### Q4：文档看不懂怎么办？

**A：** 
1. 动手敲代码，不要只看不练
2. 去 Vue 官方文档看具体例子
3. 去 B 站看视频教程
4. 在群里提问

### Q5：如何检验学习成果？

**A：** 
1. 能独立完成 CRUD 页面
2. 能看懂 Element Plus 组件文档
3. 能封装简单的工具函数
4. 能使用 Git 管理代码
