## 一、如何学习前端

### 学习方法论

作为过来人，我建议你按照以下步骤学习前端：

```
第一步：打牢基础        → HTML + CSS + JavaScript
       ↓
第二步：掌握框架        → Vue3 / React（选一个深入）
       ↓
第三步：学习配套工具      → Vite + Axios + Router + State
       ↓
第四步：积累组件库经验     → Element Plus / Ant Design
       ↓
第五步：学习工程化       → Git + ESLint + 项目构建
       ↓
第六步：做项目！做项目！做项目！
```

### 核心学习技巧

1. **不要死记硬背**：前端知识点太多，要理解原理
2. **多看源码**：开源项目的代码是最好的老师
3. **动手实践**：看 10 遍不如动手写 1 遍
4. **学会调试**：Chrome DevTools 是你最好的朋友
5. **分解问题**：复杂页面拆分成小组件

## 四、实战页面解析

### 4.1 登录页面

**文件位置**: `views/login.vue`

#### 页面效果
- 输入账号密码
- 点击登录按钮
- 成功后跳转到首页

#### 技术要点

##### 1. 表单组件 `<el-form>`

```vue
<!-- 绑定数据 model，绑定校验规则 rules -->
<el-form ref="loginRef" :model="loginForm" :rules="loginRules">
  <!-- el-form-item 需要 prop 属性指定校验字段 -->
  <el-form-item prop="username">
    <el-input v-model="loginForm.username" />
  </el-form-item>
</el-form>
```

**关键点解释**：
- `ref="loginRef"` - 给表单起个名字，后面可以用 `proxy.$refs.loginRef` 访问它
- `:model="loginForm"` - 绑定表单数据对象
- `:rules="loginRules"` - 绑定校验规则
- `prop="username"` - 指定校验哪个字段

##### 2. 响应式数据

```javascript
// 使用 ref() 创建响应式数据（适合基本类型）
const loginForm = ref({
  username: "super",    // 默认值
  password: "admin123",
  rememberMe: false
})

// 使用 reactive() 创建响应式数据（适合对象）
const loginRules = reactive({
  username: [{ required: true, trigger: "blur", message: "请输入账号" }],
  password: [{ required: true, trigger: "blur", message: "请输入密码" }]
})
```

**ref vs reactive 对比**：

| 特性 | ref | reactive |
|------|-----|----------|
| 适用类型 | 基本类型 + 对象 | 对象/数组 |
| 访问方式 | `.value` | 直接访问 |
| 示例 | `count.value` | `obj.name` |

##### 3. 登录方法

```javascript
function handleLogin() {
  // 1. 校验表单
  proxy.$refs.loginRef.validate(valid => {
    if (valid) {
      // 2. 调用登录 API
      userStore.login(loginForm.value).then(() => {
        // 3. 成功后跳转
        router.push('/index')
      })
    }
  })
}
```

**流程图**：
```
用户点击登录
    ↓
validate() 校验表单
    ↓
通过 → userStore.login() 调用接口
    ↓
成功 → router.push() 跳转页面
    ↓
失败 → 提示错误信息
```

##### 4. Pinia 状态管理

```javascript
// 导入 user store
import useUserStore from '@/store/modules/user'

// 获取 store 实例
const userStore = useUserStore()

// 调用 store 中的 action
userStore.login(loginForm.value)
```


```vue

  <el-table-column label="客户名称" prop="name" />

  </el-table-column>
  
  <!-- 根据值显示不同样式 -->

        {{ scope.row.customerType === 1 ? '个人' : '企业' }}
      </el-tag>

  </el-table-column>

```

**`scope` 对象**：
```javascript
scope = {
  row: { id: 1, name: '张三', ... },    // 当前行数据
  $index: 0,                            // 行索引
  column: { label: '姓名', prop: 'name' }, // 列配置
  store: { ... }                        // 表格内部状态
}
```

##### 4. 计算属性

```javascript
// 根据现有数据计算出统计值
const personalCount = computed(() => {
  return publicSeaList.value.filter(c => c.customerType === 1).length
})
```

**computed vs methods**：

| 特性 | computed | methods |
|------|----------|---------|
| 调用 | 自动 | 手动调用 |
| 缓存 | 有（依赖不变时不重新计算） | 无（每次调用都执行） |
| 适用 | 基于响应式数据的计算 | 执行操作/业务逻辑 |

**关键点**：

```javascript
const isEdit = ref(false)

function handleSign() {
  isEdit.value = false
  reset()  // 重置表单
  dialogTitle.value = "签署合同"
  // 自动生成合同编号
  generateNo().then(response => {
    form.value.contractNo = response.data
    dialogVisible.value = true

}

// 点击编辑
function handleEdit(row) {
  isEdit.value = true
  dialogTitle.value = "编辑合同"
  getContract(row.id).then(response => {
    form.value = response.data || response
    dialogVisible.value = true
  })

// 提交表单

  proxy.$refs["formRef"].validate(valid => {
    if (valid) {
      if (form.value.id != undefined) {
```

## 五、API 接口详解

### API 文件结构

```javascript
// 路径：api/sales/publicSea.js
import request from '@/utils/request'

// 获取公海客户列表
export function getPublicSeaPage(query) {
  return request({
    url: '/customer/public-sea/page',  // 请求路径
    method: 'get',                      // 请求方法
    params: query                      // 查询参数
  })
}

// 转移客户
export function transferCustomer(data) {
  return request({
    url: '/customer/public-sea/transfer',
    method: 'put',                     // PUT 方法（更新）
    data: data                          // 请求体数据
  })
}

// 获取销售代表列表
export function listSalesReps() {
  return request({
    url: '/sysUser/sales-reps',
    method: 'get'
  })
}
```

### HTTP 方法对照表

| 方法 | 语义 | 场景 |
|------|------|------|
| GET | 获取数据 | 查询列表、详情 |
| POST | 创建数据 | 新增、提交 |
| PUT | 更新数据 | 修改全部字段 |
| PATCH | 更新数据 | 修改部分字段 |
| DELETE | 删除数据 | 删除记录 |

### 请求封装（request.js 核心逻辑）

```javascript
// 创建 axios 实例
const service = axios.create({
  baseURL: '',      // 基础 URL
  timeout: 10000    // 超时时间 10 秒
})

// 请求拦截器：发请求前处理
service.interceptors.request.use(config => {
  // 添加 token
  if (getToken()) {
    config.headers['Authorization'] = 'Bearer ' + getToken()
  }
  return config
})

// 响应拦截器：收到响应后处理
service.interceptors.response.use(res => {
  const code = res.data.code || 200
  if (code === 200) {
    return res.data  // 返回数据部分
  } else if (code === 401) {
    // 登录过期，重新登录
  }
  // 其他错误抛出异常
  return Promise.reject(new Error(msg))
})
```

### 微服务路由配置

```javascript
// 根据 URL 前缀路由到不同服务
function getBaseURL(url) {
  if (url.startsWith('/auth/')) return ''              // 认证服务
  if (url.startsWith('/sysUser')) return '/auth/api'   // 用户服务
  if (url.startsWith('/sysDict')) return '/system/api' // 系统服务
  if (url.startsWith('/contract')) return '/sales/api'  // 销售服务
  if (url.startsWith('/loanAudit')) return '/finance/api' // 财务服务
  return '/dev-api'
}
```
