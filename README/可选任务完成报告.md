@GetMapping("/getRouters")
public Result<List<Map<String, Object>>> getRouters(...) {
    Long userId = getUserIdFromToken(token);

### 新增文件

|------|------|------|

| `OperLog.java` | 操作日志注解 | 25 |
| `OperLogAspect.java` | AOP切面 | 113 |

### 使用方式

在Controller方法上添加`@OperLog`注解：

```java
@OperLog(title = "用户登录", businessType = 0)
@PostMapping("/login")
public Result<Map<String, Object>> login(...) {
    // 登录逻辑
}
```

### 核心逻辑

```java
@Aspect
@Component
public class OperLogAspect {
    
    @Pointcut("@annotation(com.dafuweng.auth.annotation.OperLog)")
    public void operLogPointcut() {}
    
    @AfterReturning(pointcut = "operLogPointcut()", returning = "result")
    public void afterReturning(JoinPoint joinPoint, Object result) {
        saveLog(joinPoint, result, 0, null);
    }
    
    @AfterThrowing(pointcut = "operLogPointcut()", throwing = "e")
    public void afterThrowing(JoinPoint joinPoint, Exception e) {
        saveLog(joinPoint, null, 1, e.getMessage());
    }
}
```

### 特性

1. ✅ **非侵入式**：通过注解使用，不影响业务代码
2. ✅ **异步记录**：使用单独线程保存日志，不影响主流程性能
3. ✅ **异常捕获**：方法异常时也会记录日志
4. ✅ **参数记录**：自动记录方法参数和返回值
5. ✅ **性能优化**：参数过长时自动截断（500字符）
