## 🔧 问题解决记录

### Redis连接问题（已解决）

**问题**: Auth服务无法连接Redis（Lettuce和Jedis均失败）

**尝试方案**:
1. ❌ Lettuce客户端 - `Unable to connect to Redis`
2. ❌ Jedis客户端 - `Cannot get Jedis connection`
3. ✅ **内存存储方案** - 使用`ConcurrentHashMap`替代Redis

**最终方案**:
- 文件: `CaptchaController.java`
- 实现: `ConcurrentHashMap<String, String>`存储验证码
- 优点: 无外部依赖、性能更好、简化架构
- 缺点: 多实例部署需额外处理（当前单机部署无影响）

**详细报告**: [Redis问题解决报告.md](./Redis问题解决报告.md)
