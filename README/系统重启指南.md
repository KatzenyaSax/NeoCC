## ⚠️ 常见问题

### 问题1: Maven依赖下载失败

**症状**: 编译时提示找不到 `easy-captcha` 或 `springdoc-openapi`

**解决方案**:

```bash
# 清理Maven缓存
rm -rf ~/.m2/repository/com/github/whvcse
rm -rf ~/.m2/repository/org/springdoc

# 重新下载
mvn clean install -U
```

---

### 问题2: Redis连接失败

**症状**: Auth服务启动时报Redis连接错误

**解决方案**:

```bash
# 检查Redis是否运行
docker-compose ps | grep redis

# 如果未运行，启动Redis
docker-compose up -d neocc-redis

# 检查Redis配置
docker-compose exec redis redis-cli ping
# 预期返回：PONG
```

---

### 问题3: Gateway XSS过滤器报错

**症状**: Gateway启动时提示类找不到

**解决方案**:

```bash
# 重新编译Gateway
cd gateway
mvn clean package -DskipTests

# 重启Gateway
docker-compose up -d --build neocc-gateway
```

---

### 问题4: 前端页面标题未更新

**症状**: 浏览器仍显示"若依管理系统"

**解决方案**:

```bash
# 清除前端缓存并重新构建
cd ruoyi-ui
rm -rf node_modules/.vite
npm run build

# 重启Nginx
docker-compose restart neocc-nginx
```
