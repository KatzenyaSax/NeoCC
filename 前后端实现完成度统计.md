## 二、已完成的功能（前后端均实现）

### 2.1 销售模块 - 客户管理

**后端**：`CustomerController.java` + `CustomerService`
- `GET /api/customer/{id}` — 查询单个客户
- `GET /api/customer/page` — 分页查询
- `GET /api/customer/listBySalesRepId/{id}` — 按销售代表查询
- `GET /api/customer/listByStatus` — 按状态查询
- `POST /api/customer` — 新增客户
- `PUT /api/customer` — 修改客户
- `DELETE /api/customer/{id}` — 删除客户

**前端**：`views/sales/customer/index.vue` + `api/sales/customer.js`
- 分页列表（搜索、排序）
- 新增对话框（名称、电话、身份证、状态）
- 修改对话框
- 删除确认

**对接状态**：✅ 完全对接，字段一致

---

### 2.2 销售模块 - 合同管理

**后端**：`ContractController.java` + `ContractService` + `ContractSignService`
- `GET /api/contract/{id}` — 查询单个合同
- `GET /api/contract/getByContractNo/{no}` — 按合同编号查询
- `GET /api/contract/page` — 分页查询
- `GET /api/contract/listBySalesRepId/{id}` — 按销售代表查询
- `GET /api/contract/listByStatus` — 按状态查询

- `DELETE /api/contract/{id}` — 删除合同

**前端**：`views/sales/contract/index.vue` + `api/sales/contract.js`
- 分页列表（搜索合同编号、状态标签）
- 新增/修改对话框
- 签署合同操作（status=0 时显示）
- 删除确认

**对接状态**：✅ 完全对接，签署流程完整

### 2.3 财务模块 - 贷款审核

**后端**：`LoanAuditController.java` + `LoanAuditService`
- `GET /api/loanAudit/page` — 分页查询
- `GET /api/loanAudit/{id}` — 查询详情
- `GET /api/loanAudit/getByContractId/{id}` — 按合同ID查询
- `POST /api/loanAudit` — 新增
- `PUT /api/loanAudit` — 修改
- `DELETE /api/loanAudit/{id}` — 删除
- `POST /api/loanAudit/{id}/receive` — **接收申请**
- `POST /api/loanAudit/{id}/review` — **审核**
- `POST /api/loanAudit/{id}/submit-bank` — **提交银行**
- `POST /api/loanAudit/{id}/bank-result` — **银行结果**
- `POST /api/loanAudit/{id}/approve` — **批准放款**
- `POST /api/loanAudit/{id}/reject` — **拒绝**

**前端**：`views/finance/loan-audit/index.vue` + `api/finance/loanAudit.js`
- 分页列表（状态筛选、状态标签）
- 查看详情对话框
- 批准操作（填写实际放款金额/利率）
- 拒绝操作（填写拒绝原因）

**对接状态**：⚠️ 部分对接
- ✅ 已对接：列表、详情、批准、拒绝
- ❌ 未对接：receive、review、submit-bank、bank-result 接口（前端 API 已定义但页面无对应按钮/操作）

## 四、特殊说明：RuoYi 系统管理适配层

以下接口属于**为 RuoYi 前端框架而添加的适配层**，返回的是**模拟/空数据**，不是真实业务逻辑：

| 接口 | 文件 | 说明 |
|------|------|------|
| `/system/user/**` | `RuoyiSystemController.java` | 返回空列表，非真实用户数据 |
| `/system/role/**` | `RuoyiSystemController.java` | 返回空列表 |
| `/system/menu/**` | `RuoyiSystemController.java` | 返回空列表 |
| `/getRouters` | `RuoyiAdapterController.java` | 返回硬编码菜单 |
| `/captchaImage` | `RuoyiAdapterController.java` | 关闭验证码 |
